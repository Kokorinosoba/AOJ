# 速度が遅くなる主な原因
* for文をネストする
* `list.pop(0), list.insert(0, x), del list[0]`をする
    * 特にリストのサイズが大きいとき
    * `A = A[1:]`とかはもっと遅い
    * なのでできれば`A = A[:n] + A[n+1:]`も使わないほうがいい
* `if a in A`を使うときに`A`がリストになっている
    * -> セットにしたほうが良い
* その他

# 速度向上のための構文
* リスト内包表記
* `for i in [0]*n`
    * ちょっと早い
* collections.dequeを使う
* 標準入力を全部読み込んでから処理をする
* sys.stdin.readlineをinputの代わりに使う
